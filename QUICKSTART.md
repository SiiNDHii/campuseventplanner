## For Students

### Discover & Register
1. Browse the **Events** page to see all published campus events
2. Use the search bar and category filters to narrow down
3. Click an event card to see full details
4. Click **Register** to attend

### Track Your Events
- Visit **My Events** to see all events you're registered for
- Upcoming events are highlighted at the top
- View event details and organizer announcements

### Get Reminders
- Automatic reminders ~24 hours and ~1 hour before each event
- Check **Notifications** to see all alerts
- Unregister anytime to stop receiving reminders

### Share Feedback
- After an event ends, go to the event page
- Rate the organization, venue, timing, and experience (1-5 stars)
- Optionally add comments to help organizers improve

### Report Issues
- On any event page, you can report incidents (late start, venue problems, poor management)
- Choose to submit anonymously or with your name

## For Organizers

### Create & Manage Events
1. Go to **Dashboard** (organizer-only view)
2. Click **New Event** to create
3. Fill details: title, description, venue, date/time
4. Save as draft or publish immediately
5. Edit anytime before or after the event

### View Participants
- Click **Manage** on any event
- Go to **Participants** tab
- See all registered attendees and registration times
- Export to CSV (coming soon)

### Create Tasks
- On event management page, click **Tasks**
- Create team tasks with:
  - Name and assignment
  - Deadline
  - Status tracking (PENDING/DONE)
- Assign to team members for accountability

### Send Announcements
- On **Participants** page, send messages to all registrants
- Great for:
  - Venue changes or delays
  - Important dos and don'ts
  - Check-in instructions
  - Day-of updates

### Track Feedback & Incidents
- Go to **Insights** to see:
  - Event ratings (1-5 stars)
  - Feedback comments
  - Incident reports by category
  - Comparison across all your events
- Use to improve future events

## FAQ

**Q: Can I change my password?**  
A: Currently, password reset is not available. Contact your admin to reset.

**Q: Can I export participant lists?**  
A: CSV export is coming soon. For now, screenshot or manual export.

**Q: Are incident reports confidential?**  
A: Yes! You can submit anonymously. Only organizers see your report.

**Q: Can organizers see who gave feedback?**  
A: No. Feedback is anonymous unless you choose to sign it.

**Q: What if I register by mistake?**  
A: Go to the event page and click Unregister. No penalties.

**Q: Will my attendance be tracked?**  
A: Feedback submission indicates you attended, but automatic check-in is not yet implemented.
